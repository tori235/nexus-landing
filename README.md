# Nexus — The Operating System for Every Business

A modern, fully responsive landing page for Nexus, an all-in-one business platform. Built with pure HTML, CSS, and vanilla JavaScript — no frameworks or build tools required.

---

## Features

- **Responsive Design** — Adapts seamlessly from mobile to desktop
- **CSS Animations** — Scroll-triggered reveals, hover effects, and micro-interactions
- **Interactive Dashboard Mockup** — Animated metrics and chart visualization in the hero section
- **Pricing Tiers** — Three clear plans with feature comparisons
- **Social Proof** — Testimonials, trust badges, and client logos
- **Smooth Navigation** — Fixed navbar with smooth scroll and backdrop blur
- **Dark/Light Contrast** — Alternating section themes for visual rhythm

---

## Tech Stack

| Technology | Purpose |
|------------|---------|
| HTML5 | Semantic structure |
| CSS3 | Custom properties, Grid, Flexbox, animations |
| Vanilla JavaScript | Intersection Observer for scroll animations |
| Google Fonts | Syne (display) + DM Sans (body) |
